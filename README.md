### 코딩 방법
1. Gitpod 버튼을 누르세요
2. 에디터가 열리면 "src/routes/index.svelte"파일을 여세요
3. 새 창으로 뜬 페이지를 보면서 JS/HTML/CSS 코딩을 하시면 됩니다
4. 수정된 내용 업로드

### 업로드 방법

1. 저장소에 업로드할 파일 지정
```bash
git add .
```

2. 지정 파일 로컬 저장소에 업로드
```bash
git commit -m "commit message"
```

3. 로컬 저장소 데이터를 원격 저장소로 업로드
```bash
git push
```
